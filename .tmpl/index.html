Installation

These binary distributions are built off of the llvmNG branch. They will require clang, opt and llc from llvm-5.0 in PATH as well as the bootstrapped toolchain-wrapper in PATH.

The mobile haskell hackage overlay can be used similar to the one head.hackage overlay. Compare the how-to-use instructions.

NEW There is now a preliminary user guide.

Filing Issues & Getting Help

Please file issues against the hackage-overlay issue tracker on GitHub.
